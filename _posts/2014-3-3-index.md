# CoderDojo とは
```
CoderDojo は7〜17歳の子どもを対象にしたプログラミング道場です。2011年にアイルランドで始まり、世界では75カ国・1,400の道場、日本では全国に108以上の道場があります。
```
詳しくは[こちら](https://coderdojo.jp/)
